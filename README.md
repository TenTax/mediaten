# Mediaten

### Используемые технологии

`vue@2` `js` `scss` `eslint`

### Описание

Выполнение тестового задания.

### Настройка и запуск

```bash
# Установка зависимостей
yarn install

# Запуск приложения в режиме разработки
yarn serve

# Запуск линтера
yarn lint

# Сборка приложения
yarn build

# Деплой на gh-pages
yarn deploy
```
